# analytics-portfolio

## About
A collection of data analytics projects and exercises covering SQL, Python, 
Pandas, Excel, Power BI, and ML. Built as part of my journey into logistics 
digitalization and IT controlling.

## Projects

### 1. Shipping Cost Calculator (Python)
A Python-based shipping cost calculator with tiered weight pricing, 
international distance multiplier, input validation, and a loop for 
multiple calculations.

### 2. JavaScript Basics
Core JavaScript concepts practiced through logistics-themed examples, 
covering variables, conditions, functions, arrays, loops, and objects.

### 3. DOM Shipping Calculator (HTML + JavaScript)
An interactive web application built with HTML and JavaScript that takes 
a shipment weight as input, calculates the shipping cost using DOM 
manipulation, and displays the result on the page with input validation.

### 4. OOP Basics (Python)
Object-Oriented Programming concepts in Python using logistics-themed 
examples. Covers classes, objects, inheritance, encapsulation, getters, 
setters, @property, dunder methods, and instance/class/static methods.

## Skills Covered
